4 Doors

4 Doors

Date: Apr - July 2017
Role: Artist, Designer and Programmer
Group Size: 4
Project Length: 6 weeks
Setting: Project for Sound Design for Video Games class
Engine: Unity
Description: Sound-based horror game where you're trapped in a system of rooms, listening to what's behind the doors to try to escape before you're caught by what's chasing you.

Zomboat

Zomboat

Date: Jan 2017
Role: Artist, Designer
Group Size: 7
Project Length: 3 days
Setting: Global Game Jam @UCI
Engine: Unity w/ HappyFunTimes API
Description: A game where players connect to and interface with the game by going to a url on their mobile devices. All players are stuck on a boat with a small propotion of the players playing as zombies. The zombies will try to infect everyone on the ship before the time limit is over.
