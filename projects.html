Projects

This is the list of game projects I have worked on, in reverse chronological order. There are a few class projects but the rest are club/hobbyist in nature. You may be able to spot the break between my student days and my working days...

Need for Pee[2025](github,itch.io): An urgent visual novel set in a supermarket, in Unity
REALFISHING[2024](github): A realistic fishing game in Defold
Ride in the Rain[2024](github): A networked cooperative cycling game in Godot
Neck to Neck[2022](github): A cooperative climbing game in Gamemaker
Flowers in the Sky[2021](itch.io): Made for Touhou Fan Game Jam 7, a danmaku game reminiscent of Touhou Perfect Cherry Blossom stage 4, in Unity

Zombie Santa

Zombie Santa

Engine: Unity
Role: Artist, Designer, Programmer
Group Size: 3
Project Length: 3 days

Zomboat

Zomboat

Date: Jan 2017
Engine: Unity w/ HappyFunTimes API
Role: Artist, Designer
Group Size: 7
Project Length: 3 days
Setting: Global Game Jam @UCI
Description: A game where players connect to and interface with the game by going to a url on their mobile devices. All players are stuck on a boat with a small propotion of the players playing as zombies. The zombies will try to infect everyone on the ship before the time limit is over.
