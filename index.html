Past Projects(Reverse Chronological):

4Doors (2017):
A simple horror game created using Unity as a group project for a Sound Design for Video Games class.

Zomboat (2017):
A unique Unity game where anyone with a phone can join using HappyFunTimes API. After players are done joining, one player turns into a zombie and must attempt to spread the infection throughout the boat before the time limit is over.

Space Wizards (2016-2017):
Created using Unity as a group project for the UCI Computer Game Science capstone course. It's a controller(x-box/ps/etc.)-based fighting game where you are a wizard trying to push other wizards off the stage using a variety of spells that you can pick up.

Killer Self (2016):
A failed attempt during a casual game jam at making a "second-person shooter" using Unreal Engine (blueprints...) as we couldn't get networking to work.

100 Days Till Doomsday (2015-2016):
A UCI Video Game Development Club project that I attempted to lead. A turn-based tactical strategy game on Unity that was never able to reach completion.

Quaternion: Too Drunk for a Name: (2015)
A top-down fighting game made with Unity during a casual game jam. Client-server-networked barfighting.

Pokemans (2015):
A pokemon-like project using Pygame, where instead of fighting with Pokemon you fought with Men. Weird dystopian background story.

Guesstimate (2015):
Game made at the UCI Winter Game Jam that won the IEE 1st place award that year. Based off the theme "pick a side" the player must guess which side of the screen has the shapes that answer the question.

Autumn (2014-2015):
My first VGDC project; a top down shooter on Unity using time elements.

Tetris Buddies (2014):
Casual game jam project using Pygame. P2P-networked tetris.

Bill Game (2013):
Game made for history class in high school using GameMaker. Multi-level game that follows the process of a bill becoming a law.
