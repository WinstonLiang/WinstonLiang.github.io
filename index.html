Past Projects(Reverse Chronological):

4Doors (2017):
A simple horror game created using Unity as a group project for a Sound Design for Video Games class.

Zomboat (2017):
A unique Unity game where anyone with a phone can join using HappyFunTimes API. After players are done joining, one player turns into a zombie and must attempt to spread the infection throughout the boat before the time limit is over.

Space Wizards (2016-2017):
Created using Unity as a group project for the UCI Computer Game Science capstone course. It's a controller(x-box/ps/etc.)-based fighting game where you are a wizard trying to push other wizards off the stage using a variety of spells that you can pick up.

Killer Self (2016):
A failed attempt during a casual game jam at making a "second-person shooter" using Unreal Engine (blueprints...) as we couldn't get networking to work.
